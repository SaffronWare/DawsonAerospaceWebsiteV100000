Add a project

Projects added here are saved in this browser and appear immediately on the public Current or Future Projects sections, filterable by division.

If you attach a file, it's committed directly into this repo under assets/uploads/projects/ via a serverless upload function — the image itself is real and shared with everyone. The rest of the project's details are still stored locally via localStorage, so they'll only show up in this browser until a full backend is connected.

Showing projects added from this browser. Seed/demo projects that ship with the site are not listed here and can't be deleted from this page.

Add a sponsor

Sponsor logos upload to GitHub the same way project images do, then appear live in the Partners section on the homepage.

Logo files are committed into this repo under assets/uploads/sponsors/, so once added they're visible to every visitor — not just this browser. Sponsor name, tier, and link are still stored locally for now.

Sponsors added from this browser. Delete removes them from this browser's list and the public site here, but the logo file itself stays in the repo.
